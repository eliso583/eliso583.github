CONEXIÓN

La conexión es el enlace que se establece entre el emisor y el receptor a través del que se envía el mensaje.

CONECTIVIDAD

La conectividad es la capacidad de los dispositivos y sistemas de conectarse y compartir información.

TIPOS DE CONECTIVIDAD

    FIJA
    • Fibra óptica:Transmite datos mediante finos hilos de plástico o vidrio, envian información mediante pulsos de luz.
    • ADSL:Es un cable telefónico que transporta datos y se sustenta sobre línea telefónica.
    • Ethernet:Este tipo de cables envian paquetes de datos que presentan diversas ventajas, como: mayor seguridad, conexiones rápidas,etc.
    INALÁMBRICA
    • Wireless Personal:Es una red con un alcance de escasos metros, se identifica como área personal, como el bluetooth.
    • Wireless Local:Tiene un mayor alcance, incluyen conexiones más conocidas como wi-fi al igual que la radio.
    • Wireless Metropolitan:El alcance de este puede ser de 50km y permite establecer conexiones entre ubicaciones.

DIFERENCIA ENTRE CONEXIÓN Y CONECTIVIDAD

La diferencia entre conexión y conectividad es que la conexión es un vínculo entre dos o más cosas, mientras que la conectividad es la capacidad de establecer esa conexión.
